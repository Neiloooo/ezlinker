# 全局状态码

## 1.通用状态码

| 状态码 | 含义                                                  |      |
| ------ | ----------------------------------------------------- | ---- |
| 200    | HTTP请求执行成功后返回                                |      |
| 202    | 异步任务创建成功                                      |      |
| 400    | HTTP请求执行失败后返回                                |      |
| 401    | 表示用户没有权限（令牌、用户名、密码错误）            |      |
| 402    | Token异常            |      |
| 403    | 表示用户得到授权（与401错误相对），但是访问是被禁止的 |      |
| 404    | 资源不存在                                            |      |
